个人博客搭建

学习视频手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo_哔哩哔哩_bilibili

测试环境：win10+node.js

使用Hexo框架

  • 流程
    • 安装node Node.js (nodejs.org)
    • #查看版本
      • node -v
      • npm -v
      • cnpm -v
      • hexo -v
    • npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器
    • npm install --save hexo-deployer-git　　　#在blog目录下安装git部署插件
    • mkdir bolg #创建博客目录
    • hexo init #初始化博客
    • hexo clean #清理
    • hexoｓ＃启动博客
    • http://localhost:4000/ #本地访问地址
    • 修改配置文件_config.yml
      • deploy:
        type: git
        repo: https://github.com/xxx/xxx.github.io.git
        branch: master
  • 部署到github前要设置
    • git config –global user.email “xxx”
    • git config –global user.name “xxx”
    • hexo d　　＃部署到github
    • Hexo (joshua-yzx.github.io)
  • 问题：
    • 没安装git
      • git安装教程及环境配置 - grigeorge - 博客园 (cnblogs.com)
    • 进度一直卡在sill install loadAllDepsIntoIdealTree
      • 切换下载源 npm config set registry https://registry.npm.taobao.org
      • 是否换源成功：npm config get registry
